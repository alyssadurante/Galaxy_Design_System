## Overview

Select lets people choose one or more items from a list of menu items. It’s usually found in forms or inline edit.

### When to use:

- Order your list of options in a way that will make the most sense. This could be by the most commonly selected option or numerically.
- Avoid putting things in alphabetical order because it isn’t localization-friendly.
- Don’t overwhelm the user with too many options.

1. **Field Label**: Should indicate what information the field requires. The label is usually left-aligned above the input area.
2. **Text Field**: The field people click in to receive the options they can choose from. Once selected, the option will be shown in the text input.
3. **Option**: Choice for the user, shown with other choices in a menu.
4. **Menu**: Contains the list of all selectable items. These can be grouped into sub-categories under headings.
