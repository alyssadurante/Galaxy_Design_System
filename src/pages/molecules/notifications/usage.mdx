1. **Icon**: Shows the status of what the notification is for.
2. **Title**: Provides a description of what the notification is representing.
3. **Subtitle (optional)**: Contains a futher description of what the notification is for.
4. **Actions (optional)**: Allows the user to perform an action.

## Behaviors

The default notification should have a timeout function that enables it to disappear after 3 seconds. The user can close out the notification manually before that if so desired.
