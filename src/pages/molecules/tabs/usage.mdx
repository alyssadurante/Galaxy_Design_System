## Overview

Tabs are an easy way to organize content by grouping similar information on the same page. This allows content to be viewed without having to navigate away from that page.

### When to use:

- Display tabular data for comparison.
- Display structured content, where each entry has the same attributes.
- Let users review, enter, edit, or filter granular data sets.
