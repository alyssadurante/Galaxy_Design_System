## Overview

A tooltip is a message box that is displayed when a user hovers over, clicks or gives focus to a UI element such as an icon, a highlighted word, or a button.

### When to use:

- Over icon buttons to help users understand the meaning or purpose of icons
- Show the full version of truncated text.
- When space is at a premium and long content cannot be displayed all at once,
  like on a mobile interface or in a side panel.
