## Overview

The slideover is used to show more information about a certain resource in the UI and potentially handle actions on the information.

### When to use:

- Display additional or supplemental information to the user about an element or item.
- Make simple, contextual tasks available.

1. **Container**: Wraps the content. 
2. **Header**: Contains the title, close button, and actions.
3. **Close Icon**: Closes the slideover.
4. **Content**: Contains a wide variety of content types and controls depending on the purpose, related content, and contextual user tasks.
5. **Overlay**: Overlays the rest of the page content so attention is on the slideover.
